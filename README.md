# SLIMBOOK Grub Theme

Grub theme for SLIMBOOK Laptops

## To install

Download, run the installer and select your laptop model

`git clone https://github.com/Kicho01/slimbook-grub.git`

`git checkout custom`

`cd slimbook-grub`

`sudo bash install.sh`

`select your laptop`

`reboot`
